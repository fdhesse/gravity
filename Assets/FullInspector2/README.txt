### Full Inspector

Congratulations! Full Inspector provides an awesome inspector experience for the full range of .NET types.

Thank you for supporting Full Inspector. I really appreciate it!

You can view the samples by extracting the "FullInspector2-Samples" Unity package that is included inside of the "FullInspector2" folder.

The documentation is freely available at jacobdufault.github.io/fullinspector/guide. I hope you find it useful! If you have any other questions, please post to the forum topic at forum.unity3d.com/threads/224270, create an issue at github.com/jacobdufault/fullinspector/issues, or send me an email at jacobdufault@gmail.com
