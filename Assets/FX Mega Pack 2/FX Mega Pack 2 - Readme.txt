FX Mega Pack is a shuriken particle system collection, packed with effects. 
This pack uses 3D mesh particles, all the effects can be customized. Colors 
can be changed directly in Unity.

Features

• 100+ Shuriken Particle Systems
• Low Poly 3D Particles
• Packed with textures and effects
• Includes Scene and preview GUI script
• Easy Drag and Drop Prefabs
• Includes particle scale tool for easy scaling
• Text is optional, simply delete the text particle system inside prefab.
• 3D Alphabet included
